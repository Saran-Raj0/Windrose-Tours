# WINDROSE TOURS Website

A beautiful travel website for WINDROSE TOURS with red and white theme.

## Logo Image

Please add your logo image file named `logo.png` with dimensions **479 x 138 pixels** in the root directory of the project. The logo will automatically display in the header of all pages.

## Features

- Responsive design (mobile, tablet, desktop)
- Red and white color theme throughout
- Service carousel with auto-slide and touch/swipe support
- Tab-based contact forms
- Admin dashboard for viewing submissions
- Form data stored in localStorage

## Pages

- `index.html` - Home page with hero section, services carousel, and achievements
- `about.html` - About page with company information
- `contact.html` - Contact page with tabbed forms
- `admin.html` - Admin dashboard to view submissions

## Usage

Simply open `index.html` in a web browser to view the website.
